Implementation of a NFS cache, backed with Parquet, for Containers that have
internal Polars Dataframes, and use the fast Polars parquet reader.
When storing data in NFS cache use PyArroewr's ParquetWriter.

# implementation
# dbcache = DBCache(Path("__cache__/nfs"),")

# decorators
# #@dbcache.data_container_cache

Run: $ uv run -m main
To test cache, expected output when done:
Reading: parquet/A_TEST_1048576.parquet...
Returning cached object: parquet/A_TEST_1048576.parquet...

The cache should be able to handle multiple clients.
Writing to cache should be done threadsafe in parallel.
If reading (cold) container data is slow (i.e. from a DB),
create a .part file during reading, and rename it to the final name when done.

Cache key: __cache__/nfs/parquet/A_TEST_1048576.parquet will not work,
if content hash(parquet/A_TEST_524288.parquet) like a SCN from Oracle,
the cache need to invalide that item and restore.
